极限运算法则

有限个无穷小的和也是无穷小
有界函数与无穷小的乘积是无穷小
有限个无穷小的乘积也是无穷小
单调有界必有极限

无穷小的比较

lim b/a = 0 ,则b是a的高阶无穷小，记作b=o(a)
lim b/a = 无穷 ,则b是a的低阶无穷小，
lim b/a = c ,c为不为0的常数，则b是a的同阶无穷小，
lim b/a^k = c ,c为不为0的常数，k>0,则b是a的k阶无穷小，
lim b/a = 1 ,b是a的等价无穷小。

间断点类型

第一类
可去间断点
跳跃间断点

第二类
无穷间断点
震荡间断点

渐近线方程

k=lim f(x)/x

b=lim f(x)-kx

可导性与连续性

可导必连续，连续不一定可导

莱布尼茨公式

(uv)(n)=ΣC(k,n)u(n-k)v(k)

函数近似公式

f(x)=f(x0)+f’(x0)(x-x0)

微分中值定理

费马引理

函数f(x)在点x0的某邻域内有定义，并且在x0处可导，如果对邻域内任意的x，有f(x)《(》) f(x0)，那么f’(x0)=0

罗尔定理

f(x)满足在闭区间[a,b]上连续，开区间(a,b)内可导，且f(a)=f(b)，那么在(a,b)内至少有一点c，使得f’(c)=0

拉格朗日中值定理

f(x)满足在闭区间[a,b]上连续，开区间(a,b)内可导，那么在(a,b)内至少有一点c，使得f’(c)(b-a)=f(b)-f(a)

柯西定理

f(x)，F(x)满足在闭区间[a,b]上连续，开区间(a,b)内可导，且F(x)不等于0，那么在(a,b)内至少有一点c，使得f’(c)/F’(c)=f(b)-f(a)/F(b)-F(a)
