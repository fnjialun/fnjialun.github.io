利用hexo搭建个人博客(小白篇)

Posted on 2018-08-19 | In 作品 | Visitors:

相信大家可能会问，现在什么时代了，谁还写博客 有疑问的话

请看这篇文章 为什么你要写博客

本期教程是相对小白而言的，大佬请自觉绕道

首先搭建过程一共分为几步
安装Git Bash
安装NodeJs
安装hexo
生成SSH并添加到github
部署项目

我们依次开始吧！！！

  1. 安装Git Bash

下载地址
安装步骤：双击下载好的exe文件，一路next就好啦

  1. 安装NodeJs

Hexo是基于nodeJS环境的静态博客，里面的npm工具很有用啊，所以还是老老实实把这玩意儿装了吧

下载地址(说明：LTS为长期支持版，Current为当前最新版)
安装步骤：反正下载好msi文件后，双击打开安装，也是一路next，不过在Custom Setup这一步记得选 Add to PATH

  1. 安装hexo

看到这么多安装，千万不要紧张，小哥哥小姐姐们一定要稳住，别怕，因为后面的东西都是在gitbash中用npm工具安装就好了。

先在电脑里创建一个文件夹（用来存放所有blog的东西）如d:/blog/hexo，
在hexo文件夹里右键点击Git Bash Here
输入安装hexo命令：npm i -g hexo 安装完毕之后
输入初始化命令：hexo init

  1. 连接到github

没账号的创建账号，有账号的看下面。
创建一个repository，名称为yourname.github.io, 其中yourname是你的github名称，按照这个规则创建才有用哦

创建完毕之后在hexo文件夹里右键点击Git Bash Here
输入 git config –global user.name “YourName” 与 git config –global user.email “YouEmail”
把这里的 YourName 跟 YourEmail 改成你自己的

之后输入 ssh-keygen -t rsa -C “youremail@example.com“ ，生成ssh，注意改成刚才输入的邮箱
之后根据提示操作，最后会形成一段代码就是你的SSH密钥

然后在你的Github首页菜单栏中点击 Settings –> Deploy keys –> Add deploy key，然后将生成的代码中的内容全选复制到 key 输入框中，名字可以随意，然后点击 Add key 完成添加公钥。

之后打开hexo的_config.yml文件，修改里面的最后

deploy:
type: git
repo: git@github.com:yourname/yourname.github.io.git
branch: master

之后在git里输入 npm install hexo-deployer-git –save

然后 hexo d -g 即可在 https://yourname.github.io/ 查看到默认的hexo博客

  1. 部署项目

新建文章：在git下输入 hexo new ‘文章名’ ，然后你就可以在source/posts路径下看到你创建的文章啦，然后用记事本打开即可进行编辑

之后输入 hexo s 可进行本地查看，在地址栏中输入 http://localhost:4000/ 即可，满足要求之后

可输入 hexo d -g 发布到你的GitHub Pages上！！

本期教程到此结束！！！
